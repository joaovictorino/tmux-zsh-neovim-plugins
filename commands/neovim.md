## Neovim

prefix = space<br>
ctrl + ww = navigate panes<br>
ctrl + wh = navigate panes<br>
prefix + s = save file<br>
prefix + q = close nvim<br>
gt = navigate tabs

## NVimTree

prefix + e = open/close<br>
a = add file<br>
d = delete file<br>
r = rename file<br>
c = copy file<br>
x = cut file<br>
p = paste file<br>
Ctrl + x = horizontal split<br>
Ctrl + t = new tab<br>
Ctrl + v = vertical split<br>
g? = help


## Others
prefix + ff = find file<br>
prefix + fs = find text
